Allows an iterator to be do an inner join with another iterator to combine their values or filter based on their keys. this trait is applied to an iterator over a map like structure

fn outer_join<I>(self, right: I) -> OuterJoinIterator<Self, I> where I: OrderedMapIterator<Key=Self::Key>

Join an ordered iterator with the right ordered iterator. The new iterator will return a key value pair for every key in either iterator. If a key is present in both iterators they will be returned together (two values). If a value is in the Right, but not the left iterator it will be return without the value in the left iterator. If the value is in the left iterator by not the right that will be return without the value from the left iterator.
