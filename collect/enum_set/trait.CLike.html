    fn to_uint(&self) -> uint;

An interface for casting C-like enum to uint and back. A typically implementation is as below.

#[repr(uint)]
enum Foo {
    A, B, C
}

impl CLike for Foo {
    fn to_uint(&self) -> uint {
        *self as uint
    }

    fn from_uint(v: uint) -> Foo {
        unsafe { mem::transmute(v) }
    }
}
